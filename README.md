# Projeto de Estudos – C# Praticando Conceitos

Projeto simples desenvolvido para fins de estudo em **C#**, como parte do curso da **Rocketseat**.

## Objetivo
Praticar conceitos básicos da linguagem C# e execução de aplicações via CLI.

## Como executar o projeto

### Pré-requisitos
- .NET SDK instalado

### Passos
1. Abra a solução do projeto
2. Execute o build:
   ```bash
   dotnet build
3. Execute a aplicação:
   ```bash
   dotnet run
